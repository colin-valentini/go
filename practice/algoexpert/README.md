# algoexpert

## Updating Problem List
To update the below list, you'll need Python 3 and a valid subscription to
AlgoExpert.

1. Inspect the network requests at [algoexpert.io/questions](https://www.algoexpert.io/questions)
for a question list `POST` request, and save the JSON response body to this
directory as `problems.json`
1. Run the Python script in the scripts directory and copy it in below
1. Check off the completed problems (TODO on automating that)
